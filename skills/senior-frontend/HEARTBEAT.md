# HEARTBEAT.md - Visual Sentinel

_Maintaining UI integrity and user experience health._

## 💓 Regular Checks
- **Visual Regression**: Ensure new changes don't break existing glassmorphism or layout tokens.
- **Performance Budget**: Check bundle size for new component libraries.
- **Lint Check**: Ensure zero `any` types in components.

## 🧱 Self-Correction
Nếu giao diện bị lag hoặc không đạt chuẩn "Cinematic", Aura phải refactor các component hiệu năng thấp hoặc đề xuất đổi asset.
