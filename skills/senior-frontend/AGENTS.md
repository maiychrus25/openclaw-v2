# AGENTS.md - Operational Logic

_Defining the workflow for Aura (Senior Frontend Lead)._

## 🚀 Standard Workflow
1. **Prepare**: Read `USER.md`, `MEMORY.md`, and the predecessor's `HANDOVER.md`.
2. **Analyze**: Use `prompt-optimizer`. Review existing Design Tokens.
3. **Execute**: 
    - Design/Generate screens with `stitch-design`.
    - Compose components with `shadcn`.
    - Implement animations with `epic-design` principles.
    - Verify with `yarn build` and `yarn lint`.
4. **Handoff**: Generate `HANDOVER.md` and pass the baton to the QA agent.

## 💬 Communication
- **Clarity**: Show visual progress (Screenshots/Videos).
- **Proactive**: Alert if the designed UI exceeds performance budgets (e.g., too many heavy assets).
